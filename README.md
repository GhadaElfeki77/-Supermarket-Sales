# -Supermarket-Sales
Supermarket Sales Dashboard project built using Excel Power Query

a Supermarket Sales Dashboard using Microsoft Excel Power Query with the objective of transforming raw sales data into meaningful insights that support business decision-making.

Step 1: Data Understanding & Cleaning by Power Query

 I started by reviewing the raw dataset to understand the structure and key variables, such as:
*Product categories and sub-categories
*Sales, quantity, and revenue
*Dates and customer-related attributes

Then, I performed data cleaning, by Power Query which enabled an automated and repeatable data preparation workflow which included:
*Standardizing date formats and numeric fields
*Ensuring all columns were analysis-ready

This step was essential to guarantee accuracy and reliability in the analysis.

Step 2: Data Analysis Using Pivot Tables

After cleaning the data, I used Pivot Tables to analyze sales performance from multiple perspectives, including:
 *Total sales by branch and city
 *Sales trends over time
 *Performance by product category
 *Comparison of revenue and quantity sold

Pivot Tables allowed me to quickly summarize large datasets and identify key patterns and trends.

Step 3: Charts & Visual Representation

To make the analysis more intuitive, I created different types of charts, such as:
 *Column and bar charts to compare sales across branches and categories
 *Line charts to visualize sales trends over time
 *Pie charts to show contribution percentages

These visuals helped translate numerical insights into easy-to-understand stories.
 
Step 4: Dashboard Design & Visualization

Finally, I combined all insights into an interactive Excel dashboard.
The dashboard includes:
 *Key Performance Indicators (KPIs) such as total sales and average revenue
 *Dynamic charts connected to Pivot Tables
 *Slicers that allow users to filter data by branch, date, or product category

The goal was to create a dashboard that is clear, interactive, and decision-focused.

 **Conclusion:
This project demonstrates my ability to:
 *Clean and prepare data effectively
 *Analyze business data using Excel Pivot Tables
 *Design professional dashboards and visualizations
 *Transform raw data into actionable insights

Overall, this dashboard provides supermarket managers with a quick and accurate overview of sales performance, helping them make informed strategic decisions.informed strategic decisions.
